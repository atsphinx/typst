==============
atsphinx-typst
==============

Generate Typst sources and PDF from Sphinx document.

Getting started
===============

.. code:: console

   pip install atsphinx-typst

.. code:: python

   extensions = [
       ...,  # Your extensions
       "atsphinx.typst",
   ]
